GoNature Park Management System
Developed as part of the Engineering Methods for Developing Software Systems course.
This project implements a client-server system for managing park visits and staff operations using Java, JavaFX, and MySQL.

Key Features:
User reservations for park visits.
Management of park parameters by park employees.
Approving or denying requests.
Generating reports for department managers.
Seamless front-end and back-end collaboration for optimal performance.
Getting Started
Steps to Run the Project:
Download the project zip file.
Extract the zip file.
Import the database files (prototypenew.sql and users.sql) to MySQL.
Open the project in Eclipse.
Export the Server as a JAR file.
Export the Client as a JAR file.
