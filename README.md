# GoNature Park Management System
**Developed as part of the Engineering Methods for Developing Software Systems course.**<br>
This project implements a client-server system for managing park visits and staff operations using **Java**, **JavaFX**, and **MySQL**.<br>

### Key Features:<br>
- User reservations for park visits.<br>
- Management of park parameters by park employees.<br>
- Approving or denying requests.<br>
- Generating reports for department managers.<br>
- Seamless front-end and back-end collaboration for optimal performance.<br>

---

## Getting Started

### Steps to Run the Project:

1. **Download** the project zip file.<br>
2. **Extract** the zip file.<br>
3. **Import** the database files (`prototypenew.sql` and `users.sql`) to **MySQL**.<br>
4. **Open** the project in **Eclipse**.<br>
5. **Export** the Server as a JAR file.<br>
6. **Export** the Client as a JAR file.<br>

---

### Server Setup

<img src="https://github.com/user-attachments/assets/2a2e148c-435c-4b33-8570-a21c15e00929" width="600" height="500" /><br>

1. Insert your **MySQL username**.<br>
2. Insert your **MySQL password**.<br>
3. Set the **port** (recommended: 5555).<br>
4. Click **Start**.<br>
5. Click **Import Users** to populate user data.<br>

2. Click **Connect**.<br>
